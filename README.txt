# ED_Practica_Final
Práctica Final de la Asignatura de Estructura de Datos - Rutas Aéreas

## Instalación
```bash
make documentacion
make clean
make rebuild
make #para ejecutar el programa
```

## Autores
- Blanca Abril González 
- David Román Castellano

Para realizar la práctica final, hemos usado la técnica de Pair Programming, en la que una
persona programaba y la otra monitorizaba lo que hacía. Esta técnica se hacía por turnos donde
el papel de cada persona rotaba cada 30 minutos. Para sincronizar el proyecto hemos usado
un repositorio privado en GitHub
